# Contributing to Floating Lyrics Widget

First off, thank you for considering contributing to the Floating Lyrics Widget! It's people like you that make open-source software such a great community. 

Whether you are fixing a bug, improving the lyric-matching algorithm, or proposing a new feature, your help is deeply appreciated.

## 🧠 Getting Started

Before you begin, please ensure you have the correct development environment set up. Because this is a native cross-platform application, you will need tools for both the frontend and the backend.

### Prerequisites
* **Node.js** (v20 or higher) and `npm`
* **Rust** (`rustup`, `cargo`)
* **OS-Specific Dependencies:** See the [Tauri Prerequisites Guide](https://tauri.app/v1/guides/getting-started/prerequisites) to ensure your system can compile native desktop apps.

1. Fork this repository and clone it to your local machine.
